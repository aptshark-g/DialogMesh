# DialogMesh 融合器 --- 工程实现文档\n\n> **文档编号**: ENGINEERING-V3.3-FUSION-006  \n> **版本**: v1.0  \n> **日期**: 2026-07-05  \n> **状态**: 工程待实现  \n> **对应算法文档**: DESIGN_V3_3_ALGORITHM.md S6  \n> **前置算法**: v3.3 算法设计 S6 --- 分阶段融合 + 全局工作空间  \n> **原则**: 融合器不做新推理, 只做已有结果的合并与冲突仲裁。四轨返回时间差一个数量级, 分批融合是核心。  \n\n---\n\n## 1. 文档目标与范围\n\n为融合器提供完整的工程实现规范。覆盖三阶段时序对齐、五级冲突优先级、简化版全局工作空间、主动矛盾消解。\n\n### 边界\n\n| 边界 | 包含 | 不包含 |\n|------|------|--------|\n| 输入 | 四轨输出 (Track-0/-1/-P/因果) | 原始用户输入 |\n| 输出 | 单轮决策 + 置信度 | 新推理/新预测 |\n| 职责 | 合并 + 排序 + 仲裁 | 认知画像更新、边权重修改 |\n\n---\n\n## 2. 架构总览\n\n### 管线位置\n\n```\nTrack-0 (算法/10ms) --+\nTrack-1 (LLM/150ms) --+---> [FusionEngine] ---> 最终决策\nTrack-P (预测/80ms) ---+       |\nCausal (因果/<1ms) ----+       v\n                        GlobalWorkspace (轨道竞争)\n```\n\n### 内部架构\n\n```\n[StageManager]\n  |-- Stage1 (10ms): Track-0 + Causal -> 硬事实\n  |-- Stage2 (80ms): Stage1 + Track-P -> 预加载\n  |-- Stage3 (150ms): Stage2 + Track-1 -> 最终\n  |-- 降级: 超时则跳过\n  v\n[ConflictResolver]\n  |-- 五级优先级: STRUCTURAL > FACTUAL > SEMANTIC > STRATEGIC > PREDICTIVE\n  |-- 主动矛盾消解: INTENT_ACTION / ENTITY_PREDICTION / CONFIDENCE_DIVERGENCE\n  v\n[GlobalWorkspace]\n  |-- track_repression_count: 连续压制 > 3 次 -> 优先级 +1\n  |-- 选择 confidence > 0.5 的最高优先级轨道输出\n  v\n最终决策\n```\n\n### 文件结构\n\n```\ncore/agent/v3_2/fusion/\n  __init__.py\n  models.py              # FusionResult, StageOutput, TrackType\n  stage_manager.py       # StageManager (三阶段)\n  conflict_resolver.py   # ConflictResolver (五级 + 矛盾)\n  global_workspace.py    # GlobalWorkspace (~100行)\n  fusion_engine.py       # FusionEngine 入口\n```\n\n---\n\n## 3. 数据模型 (models.py)\n\n### 3.1 TrackType\n\n```python\nclass TrackType(str, Enum):\n    TRACK_0 = "algo"      # 算法层\n    TRACK_1 = "llm"       # LLM 语义层\n    TRACK_P = "pred"      # 预测层\n    CAUSAL = "causal"     # 因果基地\n    STRATEGIC = "strategic"  # 策略层\n\n@dataclass\nclass TrackResult:\n    track: TrackType\n    output: dict              # 轨道输出的具体内容\n    confidence: float         # [0,1] 轨道置信度\n    latency_ms: float         # 轨道处理耗时\n    priority_level: int = 0   # 基础优先级 (5=最高)\n    repression_count: int = 0 # 被压制次数\n    is_timeout: bool = False  # 是否超时\n\n    def is_confident(self) -> bool:\n        return self.confidence > 0.5 and not self.is_timeout\n\n@dataclass\nclass StageOutput:\n    stage: int               # 1, 2, 3\n    tracks: list[TrackResult]\n    merged: dict             # 当前阶段的合并结果\n    is_final: bool = False   # 是否作为最终输出\n    latency_ms: float = 0.0\n\n@dataclass\nclass FusionResult:\n    final_output: dict       # 最终决策\n    confidence: float        # 整体置信度 [0,1]\n    dominant_track: TrackType  # 主导轨道\n    conflicts: list[dict]    # 已解决的冲突列表\n    stages: list[StageOutput] # 各阶段中间结果\n    ask_clarification: bool = False\n    latency_ms: float = 0.0\n    profile_lite: bool = False\n```\n\n---\n\n## 4. StageManager (三阶段时序)\n\n### 4.1 职责\n\n管理三阶段时序对齐: Stage1(10ms) -> Stage2(+80ms) -> Stage3(+150ms)。每个阶段产生一个中间结果, 超时则跳过。\n\n### 4.2 接口\n\n```python\nclass StageManager:\n    STAGE1_TIMEOUT = 0.010  # 10ms\n    STAGE2_TIMEOUT = 0.080  # 80ms\n    STAGE3_TIMEOUT = 0.150  # 150ms\n\n    def __init__(self):\n        self.stages: list[StageOutput] = []\n\n    async def run_stage1(\n        self, track0: TrackResult, causal: TrackResult\n    ) -> StageOutput:\n        start = time.monotonic()\n        tracks = [t for t in [track0, causal] if t is not None and not t.is_timeout]\n        merged = self._merge_hard_facts(tracks)\n        out = StageOutput(stage=1, tracks=tracks, merged=merged)\n        out.latency_ms = (time.monotonic() - start) * 1000\n        self.stages.append(out)\n        return out\n\n    async def run_stage2(\n        self, stage1: StageOutput, track_p: TrackResult\n    ) -> StageOutput:\n        if not stage1 or track_p is None or track_p.is_timeout:\n            stage1.is_final = True  # 跳过 Stage2\n            return stage1\n        merged = dict(stage1.merged)\n        merged["predicted_actions"] = track_p.output.get("predicted_actions", [])\n        merged["expected_outcomes"] = track_p.output.get("expected_outcomes", [])\n        out = StageOutput(stage=2, tracks=stage1.tracks + [track_p], merged=merged)\n        self.stages.append(out)\n        return out\n\n    async def run_stage3(\n        self, stage2: StageOutput, track1: TrackResult\n    ) -> StageOutput:\n        if not stage2 or track1 is None or track1.is_timeout:\n            stage2.is_final = True\n            return stage2\n        merged = dict(stage2.merged)\n        merged["intent"] = track1.output.get("intent", {})\n        merged["semantics"] = track1.output.get("semantics", {})\n        out = StageOutput(stage=3, tracks=stage2.tracks + [track1], merged=merged, is_final=True)\n        self.stages.append(out)\n        return out\n\n    def _merge_hard_facts(self, tracks: list[TrackResult]) -> dict:\n        """合并硬事实: STRUCTURAL > FACTUAL"""\n        result = {}\n        for t in sorted(tracks, key=lambda x: -x.priority_level):\n            if t.is_confident():\n                result.update(t.output)\n        return result\n```\n\n### 4.3 降级策略\n\n| 场景 | 降级行为 | 最终输出源 |\n|------|---------|-----------|\n| Stage1 超时 | 跳过 Stage1, 降级到纯 LLM 决策 | Stage3 或 Stage2 |\n| Stage2 超时 | Stage1 作为最终输出 | Stage1 |\n| Stage3 超时 | Stage2 作为最终输出 | Stage2 |\n| 全超时 | ask_clarification | 无 |\n\n---\n\n## 5. ConflictResolver\n\n### 5.1 职责\n\n五级冲突优先级仲裁 + 主动矛盾消解。\n\n### 5.2 五级优先级\n\n```python\nclass ConflictResolver:\n    """冲突仲裁器"""\n\n    PRIORITY_ORDER = [\n        TrackType.CAUSAL,     # 5 - STRUCTURAL\n        TrackType.TRACK_0,    # 4 - FACTUAL\n        TrackType.TRACK_1,    # 3 - SEMANTIC\n        TrackType.STRATEGIC,  # 2 - STRATEGIC\n        TrackType.TRACK_P,    # 1 - PREDICTIVE\n    ]\n\n    def resolve(\n        self, stage: StageOutput\n    ) -> tuple[TrackResult | None, list[dict]]:\n        """\n        按优先级选择主导轨道。\n        返回 (主导轨道, 冲突列表)\n        """\n        conflicts = []\n        dominant = None\n\n        for priority in self.PRIORITY_ORDER:\n            candidates = [t for t in stage.tracks if t.track == priority and t.is_confident()]\n            if not candidates:\n                continue\n            dominant = candidates[0]\n            # 检测与其他轨道的冲突\n            for other in stage.tracks:\n                if other.track != priority and other.is_confident():\n                    c = self._detect_conflict(dominant, other)\n                    if c:\n                        conflicts.append(c)\n            break\n\n        return (dominant, conflicts)\n\n    def _detect_conflict(\n        self, dominant: TrackResult, other: TrackResult\n    ) -> dict | None:\n        """检测主导轨道与另一轨道之间的矛盾"""\n        # INTENT_ACTION_MISMATCH\n        dom_intent = dominant.output.get("intent", {})\n        other_intent = other.output.get("intent", {})\n        if dom_intent and other_intent and dom_intent != other_intent:\n            return {"type": "INTENT_ACTION_MISMATCH",\n                    "dominant": dominant.track.value, "other": other.track.value}\n\n        # ENTITY_PREDICTION_MISMATCH\n        dom_ents = set(dominant.output.get("entities", []))\n        other_ents = set(other.output.get("entities", []))\n        if dom_ents and other_ents and not dom_ents.issubset(other_ents) and not other_ents.issubset(dom_ents):\n            return {"type": "ENTITY_PREDICTION_MISMATCH",\n                    "dominant": dominant.track.value, "other": other.track.value}\n\n        # CONFIDENCE_DIVERGENCE\n        if dominant.confidence > 0.8 and other.confidence > 0.8:\n            dom_out = str(dominant.output.get("decision", ""))\n            oth_out = str(other.output.get("decision", ""))\n            if dom_out and oth_out and dom_out != oth_out:\n                return {"type": "CONFIDENCE_DIVERGENCE",\n                        "dominant": dominant.track.value, "other": other.track.value}\n\n        return None\n\n    def apply_conflict_resolution(\n        self, dominant: TrackResult, conflicts: list[dict]\n    ) -> dict:\n        """应用冲突解决策略后返回最终输出"""\n        output = dict(dominant.output)\n        for c in conflicts:\n            if c["type"] == "CONFIDENCE_DIVERGENCE":\n                output["conservative_mode"] = True\n            elif c["type"] == "INTENT_ACTION_MISMATCH":\n                output["confidence_reduction"] = 0.2\n            elif c["type"] == "ENTITY_PREDICTION_MISMATCH":\n                output["needs_clarification"] = True\n        return output\n```\n\n---\n\n## 6. GlobalWorkspace (简化版)\n\n### 6.1 职责\n\n有状态轨道竞争: 连续压制 > 3 次的轨道获得优先级提升, 防止某个轨道一直被其他轨道压制。~100 行代码。\n\n### 6.2 接口\n\n```python\nclass GlobalWorkspace:\n    """简化版全局工作空间: 有状态轨道竞争"""\n\n    BASE_PRIORITY = [TrackType.CAUSAL, TrackType.TRACK_0, TrackType.TRACK_1, TrackType.STRATEGIC, TrackType.TRACK_P]\n    SUPPRESSION_LIMIT = 3\n\n    def __init__(self):\n        self.repression_count: dict[TrackType, int] = {\n            TrackType.CAUSAL: 0,\n            TrackType.TRACK_0: 0,\n            TrackType.TRACK_1: 0,\n            TrackType.TRACK_P: 0,\n        }\n        self.last_dominant: TrackType | None = None\n\n    def select_dominant(\n        self, tracks: list[TrackResult]\n    ) -> TrackResult | None:\n        """选择主导轨道: 考虑基础优先级 + 压制计数"""\n        candidates = [t for t in tracks if t.is_confident()]\n        if not candidates:\n            return None\n\n        # 计算调整后的优先级\n        scored = []\n        for t in candidates:\n            base_idx = self.BASE_PRIORITY.index(t.track) if t.track in self.BASE_PRIORITY else 99\n            boost = self.repression_count.get(t.track, 0) // self.SUPPRESSION_LIMIT\n            adjusted = (10 - base_idx) + boost\n            scored.append((adjusted, t.confidence, t))\n\n        scored.sort(key=lambda x: (-x[0], -x[1]))\n        dominant = scored[0][2]\n\n        # 更新压制计数\n        for track in self.repression_count:\n            if track == dominant.track:\n                self.repression_count[track] = 0\n            else:\n                self.repression_count[track] += 1\n\n        self.last_dominant = dominant.track\n        return dominant\n\n    def get_status(self) -> dict:\n        return {k.value: v for k, v in self.repression_count.items()}\n```\n\n---\n\n## 7. FusionEngine (入口)\n\n### 7.1 职责\n\n编排三阶段融合 + 冲突仲裁 + 全局工作空间。\n\n### 7.2 接口\n\n```python\nclass FusionEngine:\n    """融合器入口"""\n\n    def __init__(self, stage_mgr=None, resolver=None, workspace=None):\n        self.stage_mgr = stage_mgr or StageManager()\n        self.resolver = resolver or ConflictResolver()\n        self.workspace = workspace or GlobalWorkspace()\n\n    async def fuse(\n        self, track0: TrackResult | None, track1: TrackResult | None,\n        track_p: TrackResult | None, causal: TrackResult | None,\n        profile_lite: bool = False\n    ) -> FusionResult:\n        start = time.monotonic()\n        conflicts_total = []\n\n        if profile_lite:\n            # LITE 模式: 只跑 Stage1 (双轨: 算法+LLM)\n            stage1 = await self.stage_mgr.run_stage1(track0, causal)\n            dominant, conflicts = self.resolver.resolve(stage1)\n            conflicts_total.extend(conflicts)\n            if not dominant or not dominant.is_confident():\n                return FusionResult(\n                    final_output={}, confidence=0, dominant_track=TrackType.TRACK_0,\n                    conflicts=conflicts_total, stages=[stage1],\n                    ask_clarification=True, latency_ms=0, profile_lite=True\n                )\n            out = self.resolver.apply_conflict_resolution(dominant, conflicts)\n            return FusionResult(\n                final_output=out, confidence=dominant.confidence,\n                dominant_track=dominant.track, conflicts=conflicts_total,\n                stages=[stage1], latency_ms=(time.monotonic()-start)*1000,\n                profile_lite=True\n            )\n\n        # 完整三阶段\n        stage1 = await self.stage_mgr.run_stage1(track0, causal)\n        dominant1, c1 = self.resolver.resolve(stage1)\n        conflicts_total.extend(c1)\n\n        stage2 = await self.stage_mgr.run_stage2(stage1, track_p)\n        if stage2.is_final:\n            return self._finalize(stage2, conflicts_total, start)\n        dominant2, c2 = self.resolver.resolve(stage2)\n        conflicts_total.extend(c2)\n\n        stage3 = await self.stage_mgr.run_stage3(stage2, track1)\n        if stage3.is_final:\n            dominant3, c3 = self.resolver.resolve(stage3)\n            conflicts_total.extend(c3)\n            # 全局工作空间选择最终主导\n            all_tracks = list(set(stage3.tracks))\n            gs_dominant = self.workspace.select_dominant(all_tracks) or dominant3\n            out = self.resolver.apply_conflict_resolution(gs_dominant, conflicts_total)\n            return self._finalize(stage3, conflicts_total, start, gs_dominant, out)\n\n        # 降级\n        return FusionResult(\n            final_output={}, confidence=0, dominant_track=TrackType.TRACK_0,\n            conflicts=conflicts_total, stages=[stage1, stage2, stage3],\n            ask_clarification=True, latency_ms=(time.monotonic()-start)*1000\n        )\n\n    def _finalize(\n        self, stage, conflicts, start, dominant=None, output=None\n    ) -> FusionResult:\n        dom = dominant or self.resolver.resolve(stage)[0]\n        out = output or self.resolver.apply_conflict_resolution(dom, conflicts)\n        all_low = all(t.confidence < 0.5 for t in stage.tracks)\n        return FusionResult(\n            final_output=out, confidence=dom.confidence if dom else 0,\n            dominant_track=dom.track if dom else TrackType.TRACK_0,\n            conflicts=conflicts, stages=self.stage_mgr.stages,\n            ask_clarification=all_low,\n            latency_ms=(time.monotonic()-start)*1000\n        )\n```\n\n---\n\n## 8. 四轨全部低置信度的处理\n\n当所有 track 的 confidence < 0.5 时, fusion_engine 识别到 ask_clarification=True。\n此时不输出硬猜测, 而是返回信号告知上层需要向用户澄清。\n\n### 置信度汇总\n\n```python\ndef all_low_confidence(tracks: list[TrackResult]) -> bool:\n    return all(t.confidence < 0.5 for t in tracks)\n```\n\n---\n\n## 9. 测试策略\n\n### 单元测试\n\n| 测试 | 内容 | 优先级 |\n|------|------|--------|\n| test_stage_manager | 三阶段时序, 超时, 降级 | P0 |\n| test_conflict_resolver | 五级优先级, 三种矛盾检测 | P0 |\n|
