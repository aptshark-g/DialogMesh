### 4.2 我们的独特优势

1. **认知画像建模**: 元认知/发散/追踪深度/稳定性 4维EMA — 竞品无
2. **认知状态调控解析策略**: 噪声高时降低阈值, 稳定性高时扩展同义词 — 竞品无
3. **三维话题切换检测**: 时间τ+指代失调+描述变化区分"认知刷新"与"上下文断裂" — 竞品无
4. **歧义作为一等公民**: 不是猜测消解,而是显式生成Ambiguity对象+策略矩阵 — 接近Rasa但多调控维度
5. **任务图构建**: 将意图转为可执行DAG(TaskGraph), 带Fallback链 — 竞品无
6. **确定性优先+LLM兜底**: 90%+规则覆盖,<10%触发LLM, 延迟可控 — 接近Rasa但调控更细
7. **零依赖部署**: 纯Python+正则, 无torch/transformers — 竞品多依赖重型库

### 4.3 我们的相对劣势

1. **无对话管理**: Rasa有完整对话状态管理, 我们缺少
2. **无前端SDK**: 竞品有现成的前端组件(聊天窗口、按钮渲染)
3. **无多语言NLU**: Rasa支持多语言, 我们当前以中文/英文为主
4. **无训练数据管理**: Rasa有标注平台, 我们依赖规则注册
5. **无服务层**: 竞品多提供HTTP API, 我们纯库
